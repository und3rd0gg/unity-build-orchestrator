# Default BuildPipeline Config Sample

This sample contains a starter `BuildPipelineConfig.asset`.

## Import

1. Open `Window/Package Manager`.
2. Select `Build Orchestrator`.
3. Import sample `Default BuildPipeline Config`.

## After Import

- Select the imported `BuildPipelineConfig.asset` and adjust profiles/flags.
- Open `Tools/Build/Build Pipeline Manager`.

If you prefer creating a fresh config, use:
`Tools/Build/Create Build Pipeline Config`.
